# Esercizi-Logica-CSharp

### 1. FizzBuzz
**Descrizione**: Stampa numeri da 1 a 50 con sostituzioni per numeri di 3 e 5.
**Logica Applicata**: Utilizzo dell'operatore modulo '%' e la gestione delle precedenze nelle condizioni 'if/else'.

### 2. Verificatore Numeri Primi
**Descrizione**: Determina se un numero inserito dall'utente è primo o composto.
**Logica applicata**: Ottimizzazione del ciclo 'for' limitando il controllo alla radice quadrata del numero (Math.Sqrt()) per ridurre i calcoli inutili.

### 3. Invertitore di Stringhe
**Descrizione**: Prende una stringa in input e stampa il contrario.
**Logica Applicata**: Manipolazione degli indici delle stringhe e gestione dei cicli 'for' decrescenti.

## Tecnologie Utilizzate
**Linguaggio**: C# (.NET Core).
**Struementi**: Visual Studio. 
**Logica**: Algoritmi di base, parsing degli input e gestione dei tipi di dato.
