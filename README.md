# Esercizi-Logica-CSharp

In questo repository raccolgo le sfide di programmazione affrontate per consolidare le basi del linguaggio C# e del pensiero computazionale. Ogni esercizio si focalizza su un concetto specifico.

## Tecnologie Utilizzate
**Linguaggio**: C# (.NET Core).
**Struementi**: Visual Studio. 
**Logica**: Algoritmi di base, parsing degli input e gestione dei tipi di dato.
